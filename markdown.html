Markdown.

Markdown is a way of writing that allows you to style your text just a little bit. If you know HTML, it essentially allows you to do a simpler version of this.

Bold *text*
Underline _text_
Italics /text/
Horizontal Line ---
Strikethrough -text-
Small header ###text
Medium Header ##text
Large Header #text
Bullet points +text
Link (display text)[https://webaddress.com]
Code Block ```text```
Code `text`
Highlight ::text::
